# Telegram бот Лорі

Telegram бот, створений з використанням бібліотеки aiogram 3.

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть бота:
```bash
python bot.py
```

## Функціонал

- Команда `/start` - привітальне повідомлення з інлайн-кнопкою "Давай"
- Натискання на кнопку "Давай" - відображає Reply-клавіатуру з рівнями складності

## Примітка

Замініть API_TOKEN у файлі `bot.py` на ваш реальний токен бота, отриманий від @BotFather.
