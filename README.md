# Telegram бот Лорі

Telegram бот, створений з використанням бібліотеки aiogram 3.

## Встановлення

1. Встановлення залежностей:
```bash
pip install -r requirements.txt
```

2. Запуск бота:
```bash
python bot.py
```

## Функціонал

- Команда `/start` - привітальне повідомлення з інлайн-кнопкою "Давай"
- Натискання на кнопку "Давай" - відображає Reply-клавіатуру з рівнями складності
